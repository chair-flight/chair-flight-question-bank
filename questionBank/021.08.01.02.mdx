# Design, operation, system components, indications

## Learning Objectives

1. **021.08.01.02.01** -- State the tasks of the fuel system.
2. **021.08.01.02.02** -- Name the following main components of a fuel system, and
   state their location and their function: lines; boost pump; pressure valves;
   filter, strainer; tanks (wing, tip, fuselage); vent system; sump; drain;
   fuel-quantity sensor; fuel-temperature sensor.
3. **021.08.01.02.03** -- Describe a gravity fuel feed system and a pressure feed
   fuel system.
4. **021.08.01.02.04** -- Describe the construction of the different types of fuel
   tanks and state their advantages and disadvantages: drum tank; bladder tank;
   integral tank.
5. **021.08.01.02.05** -- Explain the function of cross-feed.
6. **021.08.01.02.06** -- Define the term "unusable fuel".
7. **021.08.01.02.07** -- List the following parameters that are monitored for the
   fuel system: fuel quantity (low-level warning); fuel temperature.

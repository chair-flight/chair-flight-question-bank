# Compressor and diffuser

## Learning Objectives

1. **021.11.02.02.01** -- State the purpose of the compressor.
2. **021.11.02.02.02** -- Describe the working principle of a centrifugal and an
   axial flow compressor.
3. **021.11.02.02.03** -- Name the following main components of a single stage and
   describe their function for a centrifugal compressor: impeller; diffuser.
4. **021.11.02.02.04** -- Name the following main components of a single stage and
   describe their function for an axial compressor: rotor vanes; stator vanes.
5. **021.11.02.02.05** -- Describe the gas-parameter changes in a compressor
   stage.
6. **021.11.02.02.06** -- Define the term "pressure ratio" and state a typical
   value for one stage of a centrifugal and an axial flow compressor and for the
   complete compressor.
7. **021.11.02.02.07** -- State the advantages and disadvantages of increasing the
   number of stages in a centrifugal compressor.
8. **021.11.02.02.08** -- Explain the difference in sensitivity for FOD of a
   centrifugal compressor compared with an axial flow type.
9. **021.11.02.02.09** -- Explain the convergent air annulus through an axial flow
   compressor.
10. **021.11.02.02.10** -- Describe the reason for twisting the compressor blades.
11. **021.11.02.02.11** -- State the tasks of inlet guide vanes (IGVs).
12. **021.11.02.02.12** -- State the reason for the clicking noise whilst the
    compressor slowly rotates on the ground.
13. **021.11.02.02.13** -- State the advantages of increasing the number of
    spools.
14. **021.11.02.02.14** -- Explain the implications of tip losses and describe the
    design features to minimise the problem.
15. **021.11.02.02.15** -- Explain the problems of blade bending and flapping and
    describe the design features to minimise the problem.
16. **021.11.02.02.16** -- Explain the following terms: compressor stall; engine
    surge.
17. **021.11.02.02.17** -- State the conditions that are possible causes of stall
    and surge.
18. **021.11.02.02.18** -- Describe the indications of stall and surge.
19. **021.11.02.02.19** -- Describe the design features used to minimise the
    occurrence of stall and surge.
20. **021.11.02.02.20** -- Describe a compressor map (surge envelope) with rpm
    lines, stall limit, steady state line and acceleration line.
21. **021.11.02.02.21** -- Describe the function of the diffuser.

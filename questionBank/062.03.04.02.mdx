# Modes and codes

## Learning Objectives

1. **062.03.04.02.01** -- State that the interrogator transmits its interrogations
   in the form of a series of pulse pairs.
2. **062.03.04.02.02** -- Name the interrogation modes: Mode A; Mode C; Mode S.
3. **062.03.04.02.03** -- State that the interrogation frequency and the reply
   frequency are different.
4. **062.03.04.02.04** -- Explain that the decoding of the time interval between
   the pulse pairs determines the operating mode of the transponder: Mode A:
   transmission of aircraft transponder code; Mode C: transmission of aircraft
   pressure altitude; Mode S: selection of aircraft address and transmission of
   flight data for the ground surveillance.
5. **062.03.04.02.05** -- State that Mode A designation is a sequence of four
   digits which can be manually selected from 4 096 available codes.
6. **062.03.04.02.06** -- State that in Mode C reply, the pressure altitude is
   reported in 100-ft increments.
7. **062.03.04.02.07** -- State that in addition to the information provided, on
   request from ATC, a special position identification (SPI) pulse can be
   transmitted but only as a result of a manual selection by the pilot (IDENT
   button).
8. **062.03.04.02.08** -- State the need for compatibility of Mode S with Mode A
   and C.
9. **062.03.04.02.09** -- Explain that Mode S transponders receive interrogations
   from TCAS and SSR ground stations.
10. **062.03.04.02.10** -- State that Mode S interrogation contains either the
    aircraft address, selective call or all-call address.
11. **062.03.04.02.11** -- State that every aircraft is allocated an ICAO aircraft
    address, which is hard-coded into the Mode S transponder (Mode S address).
12. **062.03.04.02.12** -- Explain that a 24-bit address is used in all Mode S
    transmissions, so that every interrogation can be directed to a specific
    aircraft.
13. **062.03.04.02.13** -- State that Mode S can provide enhanced vertical
    tracking, using a 25-ft altitude increment.
14. **062.03.04.02.14** -- State that SSR can be used for automatic dependent
    surveillance - broadcast (ADS-B).

# Performance-based navigation (PBN) principles

## Learning Objectives

1. **062.07.04.01.01** -- Define "path definition error" (PDE).
2. **062.07.04.01.02** -- Define "flight technical error" (FTE) and state that the
   FTE is the error in following the prescribed path, either by the auto-flight
   system or by the pilot.
3. **062.07.04.01.03** -- Define "navigation system error" (NSE) and state that
   the accuracy of a navigation system may be referred to as NSE.
4. **062.07.04.01.04** -- Define "total system error" (TSE) and state that the
   geometric sum of the PDE, FTE and NSE equals the TSE.
5. **062.07.04.01.05** -- State that navigation accuracy depends on the TSE.
